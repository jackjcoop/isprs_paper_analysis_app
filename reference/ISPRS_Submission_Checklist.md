# ISPRS Full Paper Submission Checklist

Use this checklist to verify your paper meets all requirements before submission.

---

## General Layout

- [ ] Paper size is A4
- [ ] Margins: 25mm top/bottom, 20mm left/right
- [ ] Font: Times New Roman, 9pt (except title)
- [ ] Single-column layout for Title, Authors, Keywords, and Abstract
- [ ] Two-column layout for Introduction through References (82mm width, 6mm spacing)
- [ ] Text is left and right justified throughout
- [ ] Paper length: 6–8 pages (including all content)

---

## Section Checklist

### 1. Title
- [ ] Present and positioned first
- [ ] Centered, bold, 12pt Times New Roman
- [ ] No identifying author information (review version)

### 2. Author(s) and Affiliation
- [ ] Positioned below title, centered
- [ ] **Review version:** Fully anonymized (no names, affiliations, or emails)
- [ ] **Review version:** No self-identifying citations (e.g., "As we showed..." → "Author (20xx) showed...")
- [ ] **Camera-ready:** Full names, institutions, addresses, and emails included
- [ ] **Camera-ready:** Multi-author papers indicate author-organization associations

### 3. Keywords
- [ ] Two blank lines after author section
- [ ] "Keywords:" label in bold
- [ ] Maximum of 6 keywords listed

### 4. Abstract
- [ ] Two blank lines after keywords
- [ ] "Abstract" label flush left, bold, followed by one blank line
- [ ] Length: 100–250 words
- [ ] Content is understandable to non-specialists
- [ ] Clearly states the scientific contribution and results

### 5. Introduction
- [ ] Present as first section of main body
- [ ] Two-column layout begins here
- [ ] Single-spaced text with one blank line between paragraphs
- [ ] Paragraphs start flush with left margin

### 6. Main Body
- [ ] Logical organization of content

**Headings:**
- [ ] Major headings: numbered, centered, bold, blank line before and after
- [ ] Subheadings: numbered, bold, flush left, blank line before and after
- [ ] Sub-subheadings: numbered, bold, flush left, text continues on same line
- [ ] Decimal numbering system used (e.g., 1., 1.1, 1.1.1)

**Footnotes:**
- [ ] Marked with consecutive superscript numbers
- [ ] Placed at bottom of page, separated by horizontal line

**Figures and Tables:**
- [ ] Placed close to their reference in text
- [ ] Large items rotated 90° with top on left side (if needed)
- [ ] Captions centered beneath each item
- [ ] Numbered consecutively (Figure 1, Figure 2... / Table 1, Table 2...)
- [ ] Copyright statements included for third-party imagery

**Equations:**
- [ ] Numbered consecutively in parentheses, flush right
- [ ] One blank line before and after each equation

**Symbols and Units:**
- [ ] SI Units (Système International) used throughout

### 7. Conclusions
- [ ] Present
- [ ] Summarizes results and scientific contribution

### 8. Acknowledgements (Optional)
- [ ] **Review version:** Fully anonymized
- [ ] **Camera-ready:** Personal acknowledgements permitted

### 9. References
- [ ] All sources cited in-text as (Author, Year)
- [ ] Listed alphabetically at end of paper
- [ ] Journal names abbreviated or written in full consistently
- [ ] Websites use DOI when available
- [ ] Software citations specify project, version, or add-on as appropriate

### 10. Appendix (Optional)
- [ ] Supporting data included only if necessary
- [ ] Total paper length still within page limits

---

## Final Review

- [ ] Spell check completed
- [ ] All figures and tables render correctly
- [ ] All references cited in text appear in reference list
- [ ] All references in list are cited in text
- [ ] Document opens correctly in Word/PDF
